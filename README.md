# Cyberpunk Calc

A retro-futuristic desktop calculator inspired by classic X11 applications,
CRT terminals, and cyberpunk aesthetics.
<img width="1278" height="936" alt="image" src="https://github.com/user-attachments/assets/5b3af204-840c-4de5-9fc9-4d73df850efa" />

## Features

- Custom X11-style window decoration
- Neon cyberpunk color palette
- CRT-inspired display with scanlines, flicker, and glow
- Keyboard and mouse input
- Standard arithmetic operations
- Square root, square, reciprocal, and percent functions
- Windows-friendly Python/Tkinter implementation
- Custom icon included
- Build script included

## Running

### From Python

```bash
python cyberpunk_calc.py
```

### Windows Batch Launcher

Double-click:

```text
run-calculator.bat
```

## Building an EXE

Install PyInstaller:

```bash
pip install pyinstaller
```

Run:

```text
build.exe.bat
```

The executable will be created in the `dist` folder.
